# Como rodar o projeto?

```sh
    # Copiar o arquivo com as configurações de ambiente 
    $ cp ./.env.example ./.env

    # Instalar pacotes composer
    $ composer install

    # Instalar pacotes NPM
    $ npm install 

    # Realizar build NPM
    $ npm run build

    # Gerar chave
    $ php artisan key:generate

    # Rodar as migrations
    $ php artisan migrate

    # Servir a aplicação no ambiente local
    $ composer run dev

```
